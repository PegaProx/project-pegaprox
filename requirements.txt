# === Encryption & Security ===
# NS May 2026 — cryptography 46.0.6/.0.7 patches CVE-2026-34073 + -39892,
# pyopenssl 26.x patches CVE-2026-27448 + -27459. argon2-cffi unchanged.
# MK Jul 2026 — bumped to 49: fido2 2.2.1 (2026-06-29) lifted its cryptography
# ceiling from <49 to <52, so we can finally close AIKIDO-2026-94131 (raw-ChaCha20
# counter overflow — the CVE the Aikido autofix PRs #550/#603 kept re-raising)
# PROPERLY, without force-dropping fido2 (WebAuthn/hardware-key 2FA). Verified: no
# removed-alias usage (PUBLIC_KEY_TYPES etc.), our crypto surface (Fernet/AESGCM/
# TripleDES-decrepit/rsa/ec/serialization) + create_app import clean on 49; PegaProx
# is Linux-only so the dropped macOS/32-bit-Windows wheels don't apply. <52 matches
